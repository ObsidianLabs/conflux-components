# ERC777

This is a smart contract implements [ERC-777](https://docs.openzeppelin.com/contracts/4.x/erc777) based on the [template](https://github.com/OpenZeppelin/openzeppelin-contracts/blob/master/contracts/token/ERC777/ERC777.sol) given by OpenZeppelin.

Notice that it can only work with **OpenZeppelin v4**.

The only modification is that this contract uses the Conflux version of IERC1820 registry at [0x88887eD889e776bCBe2f0f9932EcFaBcDfCd1820](https://confluxscan.io/address/cfx:aceju902vhx1rtf8f6h3wp1p9m8r9xj2eau7fdwk48?accountAddress=cfx%3Aaceju902vhx1rtf8f6h3wp1p9m8r9xj2eau7fdwk48).
